## About

Julya Kimaeva (b. 1988) is a visual artist based in Moscow.

Working in figurative painting, she uses cropped body fragments and everyday objects to address anonymity, distance, and memory.

Her practice explores fragmented perception and the dissonance between inner experience and external reality, revealed through ordinary things.
